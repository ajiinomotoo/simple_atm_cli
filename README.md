# ATM CLI Sederhana

Program ini mensimulasikan mesin ATM berbasis terminal menggunakan Python.

Fitur:

- Login berdasarkan PIN
- Cek saldo, tarik dan setor tunai
- Ganti PIN dan lihat riwayat transaksi
- Tambah dan hapus nasabah

Cara Menjalankan:

1. Buka terminal di folder `simple_atm_cli`
2. Jalankan: `python main.py`

Dependensi:

- Program ini menggunakan library `colorama` untuk tampilan teks berwarna di terminal.
- Install dulu dengan perintah: `pip install colorama`

(Opsional) Virtual Environment:

- Disarankan menggunakan virtual environment agar dependensi tetap terisolasi.
- Buat environment: `python -m venv .venv`
- Aktifkan:
  - Windows: `.venv\Scripts\activate`
  - Mac/Linux: `source .venv/bin/activate`
